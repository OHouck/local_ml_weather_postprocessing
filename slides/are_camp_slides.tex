\begin{frame}{Advancements in ML based Weather Forecasting have made Tailored Forecasting Models Possible}
    \begin{itemize}
        \item Traditional numerical weather prediction (NWP) models are expensive and slow to run. 
        \item ML based weather models are fast to run, but are still expensive to train. 
        \item These models predict the entire globe at once. A farmer in cares a lot about if it will rain in five days. Less about wind speeds over in the troposphere over Greenland. 

    \end{itemize}
    \pause 
    \vspace{0.5em}
    \textbf{Goal:} % Delays the appearance of the content below this line
    Demonstrate that is is possible to meaningfully improve the accuracy of relevant weather variables using a small amount of compute power and data.

    \pause 
    \vspace{0.5em}
    \textbf{Contribution:} Apply neural network post-processing model to both traditional and ML weather models, showing that it is possible to improve the accuracy of both (perhaps with some caveats).
\end{frame}
