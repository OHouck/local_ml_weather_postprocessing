\section{Background}
\begin{frame}{Advancements in ML Weather Forecasting have made Tailored Forecasting Models Possible}
    \begin{itemize}
        \item Traditional Physics Based Weather Forecasting Models Are Expensive and Slow to Run
        \item New generation of data-driven ML weather models are competitive in accuracy and can be run on a laptop
        \item New ML models are still very expensive to train and try to jointly predict the entire globe at once.
        \item If we instead just focus on weather variables most relevant to people in a specific region, can we improve accuracy?

    \end{itemize}
    \pause 
    \vspace{0.5em}
    \textbf{Goal:} % Delays the appearance of the content below this line
    Demonstrate that is is possible to meaningfully improve the accuracy of relevant weather variables using a small amount of compute power and data.
\end{frame}


\begin{frame}{Research Questions and Early Results}
\begin{itemize}
    \item Is it possible to improve forecast accuracy using easily available off the shelf tools?
    \item Can NWP and ML weather models can be corrected in the same way? 
    \item We train a simple neural network to correct both a data driven ML forecast (Pangu) and a traditional NWP forecast (IFS HRES) we find success predicting 10m wind speed but not so much 2m temperature. 
\end{itemize}

\vspace{0.5em}
\textbf{Key Contribution:} Apply neural network post-processing to both NWP and ML weather models, showing that it is possible to improve the accuracy of both with perhaps some caveats.
\end{frame}

\begin{frame}{Start by Manually Looking at Four Regions of Interest}
    \begin{itemize}
