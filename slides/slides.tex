\begin{frame}{What's new?}
\begin{itemize}
    \item Old databuild did not handle validation time correctly
    \item Rewrote databuild to be much faster and allow for using more regions. Early results for Ethiopia 
    \item Added month embeddings to unet and mlp architectures.
    \item Capable to train on multiple lead times at once using learned lead time embedding.
    \item Implemented Bayesian hyperparameter optimization and optimized parameters for 2m temperatures 1 week lead time in India.
    \item Downloaded IMD precipitation data over India
    \item Almost finished downloaded AIFS forecasts from Pedram's DSI cluster
\end{itemize}
\end{frame}
