Overview:
\begin{itemize}
    \item \textbf{Procedure:} 

    \begin{itemize}
        \item Divided the Globe into 6x6 degree regions, kept all with at least 50\% land coverage. Trained a seperate model for each region using hyperparameter values optimized for correcting Pangu 2m temperature or 10m wind speed over a 6x6 region in India. 
        \item Plot average RMSE improvement over baseline (Pangu or IFS) for each 6x6 degree patch. 
        \item Plot raw RMSE values and improvements by pixel (0.25 degree) as well.
        \item Plot RMSE improvement by patch vs distance from equator and vs standard deviation of the observation record (SDOR) for that patch.
    \end{itemize}
    \item \textbf{Takeaways:}
    \begin{itemize}
        \item Able to improve forecasts globally for variables and models
        \item For longer lead time forecasts (9 days) much higher RMSE in mid latitudes. This seems consistent with the literature: https://rmets.onlinelibrary.wiley.com/doi/full/10.1002/met.70055
        \item Weak Evidence of 2m temperature being more improved near the equator. This is also true for windspeed besides the 9 day forecast when there are more improvements in mid latitudes.
        \item Temperature prediction unaffected by SDOR, but wind speed improvements are higher in regions with higher SDOR (more variable weather).This effect is stronger for shorter term forecasts. I plan to run some simple regressions and do proper hypothesis testing on these relationships.
    \end{itemize}

    
\end{itemize}
